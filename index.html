HTML & CSS Basics Summary

HTML Summary

HTML is used to define our page content , structure and meaning . You don't use it for styling purpose. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content
  • HTML can also have Void (empty) elements like Images

CSS Summary

CSS is used for styling purpose your Page Content!

  • Styles are assigned via property-value pairs
  • To avoid code duplication, you typically use global styles
  • Alternatively you can work with external stylesheet files which you "link" to
  • When working with CSS, Concepts like "inheretnce" , "specificity" and the "box model" should be understood.
